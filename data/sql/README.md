# Customer Revenue & Retention Analytics (SQL)

## Project Overview

This project analyzes customer purchasing behavior, revenue performance, and retention metrics using SQL and Power BI.

The objective is to transform raw retail data into business insights through data cleaning, exploratory analysis, KPI development, and customer segmentation.
